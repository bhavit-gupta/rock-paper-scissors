## üöÄ Running Locally 
If you want to run or edit this project on your own machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bhavit-gupta/rock-paper-scissors.git
   cd rock-paper-scissors
   ```

2. **Install Dependencies:**
   Install Tailwind CSS packages to be able to compile the CSS.
   ```bash
   npm install
   ```

3. **Start the Tailwind Watcher:**
   If you want to make changes to the styling, run the Tailwind CLI watch command to automatically compile `src/input.css` into `src/output.css`.
   ```bash
   npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
   ```

4. **Launch the Game:**
   Simply open the `src/index.html` file in any modern web browser to play!
